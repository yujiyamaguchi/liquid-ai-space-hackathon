# HarborLens — Lean Canvas

---

## UVP

> **「衛星エッジ AI が、銀行の担保調査員より先に浮き蓋式油タンクの在庫急変を Sentinel-2 SWIR で検知し、融資リスクを自動でフラグアップする」**

---

## 1. Problem

融資銀行・保険会社が「担保物（港湾タンク・コンテナ）の現状」を独立かつタイムリーに確認する手段がない。

| 指標 | 数値 |
|---|---|
| Hin Leong 事件（2020年、シンガポール） | 約 $3.5B の不正担保融資 |
| 青島港事件（2014年） | $10B 超の不正担保 |
| 現状の監査遅延 | 週次〜月次。72時間の遅延が数百万ドルのエクスポージャー差 |

---

## 2. Customer / Early Adopter

| 層 | 組織 | 購買動機 |
|---|---|---|
| **Early Adopter** | **シンガポール拠点の中規模トレードファイナンス専門銀行** | Hin Leong 以降の担保検証需要。MAS 規制が独立担保モニタリングを事実上要求 |
| Customer Segment 2 | 貿易信用保険会社（Euler Hermes / Allianz Trade） | 損害査定の迅速化 |
| Customer Segment 3 | 港湾オペレーター | ESG レポート用途 |

## 4. Unfair Advantage

| 手法 | 検知対象 | 遅延 | コスト/拠点/月 | エッジ処理 |
|---|---|---|---|---|
| **HarborLens** | **油タンク在庫変化（floating roof 影変化）** | **軌道周期（1〜5日）** | **$300〜$800** | **あり** |
| 地上エージェント巡回 | 直接計測 | 週次〜月次 | $5K〜$20K | N/A |
| 商用衛星解析 SaaS（Kayrros / OilX 等） | RGB/SAR タンクレベル | 日次〜週次 | $2K〜$10K | なし（地上処理） |
| AIS + 船舶動静推計 | 船舶位置 | リアルタイム | $500〜$2K | N/A |

**HarborLens のみが持つ組み合わせ:**
- **オンボード処理**: 地上局なしでタンク在庫異常を軌道内で判定。証跡タイムスタンプが改ざん困難
- **LFM 文脈判断**: 「正常な在庫変動（季節的出荷）vs 詐欺的急減（担保消滅）」の解釈はルールベース差分では不可能。画像内の文脈（周辺船舶・クレーン活動・影の急変パターン）から LFM が判断
- **コスト優位**: Kayrros 等の既存 SAR ベースサービスと比較し $300〜$800/拠点/月 = 10x 低コスト（Sentinel-2 は無償）

## 8. Assumptions

1. Sentinel-2 の 10m 解像度で浮き蓋式油タンク（直径 30m 以上 = 3px 以上）の floating roof 位置変化を識別できる
2. SWIR (B11/B12) でタンク金属・液面反射の変化を検知できる
3. SimSat がシンガポール（ジュロン島）・フジャイラ港・鹿島コンビナート周辺の時系列データを十分な密度で保有している
4. 銀行コンプライアンスが「衛星証跡」を内部審査参考資料として受理する
5. **GT: AIS 外部データとの照合で検証する（自己 GT ではない）**
   - AIS で確認した船舶到着/出港日時と SimSat のタンク在庫変化を照合
   - 「船舶到着 → タンク在庫増加」「出港 → 在庫減少」という GT を構築
   - AIS ソース: Marine Cadastre 歴史 AIS / MarineTraffic API（無料枠）

## 技術スタック

```
SimSat API (Sentinel-2: B11/B12/B4/B3/B2、タンクヤード座標)
  → 油タンク ROI マスク（事前登録の floating roof タンク座標）
  → Floating roof 影長 / 反射輝度の時系列抽出: T0（ベースライン）vs T1（監視日）
  → 変化率算出（影変化 → 在庫変動推定）
  → LFM 2.5-VL-450M (LoRA FT)
     ※ LFM の役割: 「正常な在庫変動 vs 詐欺的急減（担保消滅パターン）」の文脈判断
        → アルゴリズムが検出した変化の severity と context を解釈
  → {"inventory_change": "ABNORMAL_DROP", "confidence": 0.91, "alert": true}
  → 変化ありの JSON のみダウンリンク → REST API → 銀行 TMS
```

GT: AIS 船舶到着/出港レコード × SimSat 在庫変化の時系列照合（外部検証）
